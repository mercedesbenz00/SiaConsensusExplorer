## Installation

### Web
- ``git clone https://github.com/S-anasol/SiaConsensusExplorer.git``
- ``npm i``
- ``npm run production``
- ``composer update``
- ``php artisan vendor:publish``
- ``php artisan key:generate``
- Make and edit .env config (check .env.example)
- ``php artisan migrate``
- Done!
